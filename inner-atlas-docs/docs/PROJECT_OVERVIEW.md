# Inner Atlas — Project Overview

## App Identity

**Name:** Inner Atlas  
**Tagline:** Map your inner world. Lead from within.  
**Framework:** Internal Family Systems (IFS) — Richard Schwartz  
**Tone:** Warm, professional, clinically grounded, non-pathologizing

## Three User Audiences

1. **Individual Users (Self-Led)** — adults exploring independently, no therapist
2. **Therapy Clients** — using app between sessions with an IFS therapist
3. **Therapists/Practitioners** — managing clients, session prep, worksheet assignment (Phase 5)

## Core Value Propositions

1. Only IFS app with narrative assessment-to-fog-map onboarding pipeline
2. Parts treated as real named entities — users name everything themselves
3. Self-energy tracked and developed, not just mentioned
4. App gets richer over time — living system document
5. Fog of war map creates ongoing discovery motivation
